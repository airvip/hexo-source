在 Linux 中有没有像 Windows 系统中那样的快捷方式呢？毫无疑问，在 Linux 如此强大的系统中，肯定是有的了，在 Linux 他的名称叫做软连接。

<!-- more -->

Linux 系统中有 **软链接** 和 **硬链接** 两种特殊的文件。
* 软链接可以看作是Windows中的快捷方式，可以让你快速链接到目标档案或目录。
* 硬链接则是通过文件系统的inode来产生新档名，而不是产生新档案。
